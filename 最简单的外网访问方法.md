# 🚀 最简单的外网访问方法

由于 cloudflared 安装可能遇到网络问题，这里提供**最简单的方法**：

## ⭐ 方法一：使用 localtunnel（推荐，无需安装）

**优点**：
- ✅ 无需安装额外工具（使用 npm）
- ✅ 完全免费
- ✅ 一键启动

**步骤**：

1. **确保本地服务运行**：
   ```bash
   cd /Users/bonnie/CSM_Cursor
   npm run dev
   ```

2. **新开一个终端窗口，运行**：
   ```bash
   npx localtunnel --port 3000
   ```

3. **会显示一个地址**，例如：
   ```
   your url is: https://xxxx.loca.lt
   ```

4. **告诉同事访问这个地址即可！**

**就这么简单！** 🎉

---

## 方法二：手动安装 cloudflared

如果 prefer cloudflared，可以：

1. **访问下载页面**：
   - https://github.com/cloudflare/cloudflared/releases/latest
   - 下载 `cloudflared-darwin-arm64`（你的Mac是Apple Silicon）

2. **下载后**：
   ```bash
   chmod +x cloudflared
   ./cloudflared tunnel --url http://localhost:3000
   ```

---

## 方法三：使用 ngrok（需要注册）

1. 访问 https://ngrok.com/ 注册（免费）
2. 下载安装包
3. 配置：`ngrok config add-authtoken 你的token`
4. 使用：`ngrok http 3000`

---

## ⚠️ 注意事项

1. **需要同时穿透后端**（如果登录失败）：
   ```bash
   # 再开一个终端
   npx localtunnel --port 3001
   ```
   
   然后在浏览器控制台（F12）运行：
   ```javascript
   localStorage.setItem('backend_url', '后端显示的地址');
   location.reload();
   ```

2. **地址会变化**：
   - localtunnel 每次启动地址可能不同
   - 需要重新告诉同事新地址

3. **保持服务运行**：
   - 本地服务 `npm run dev` 必须保持运行
   - localtunnel 窗口也必须保持运行
