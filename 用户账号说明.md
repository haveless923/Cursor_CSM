## 如何让同事访问系统

### 方法一：同一局域网访问（推荐）

1. **获取你的电脑IP地址**：
   - macOS: 打开"系统设置" > "网络"，查看IP地址
   - 或在终端运行：`ifconfig | grep "inet " | grep -v 127.0.0.1`

2. **确保服务器正在运行**：
   ```bash
   cd /Users/bonnie/CSM_Cursor
   npm run dev
   ```

3. **同事访问**：
   - 在浏览器输入：`http://你的IP地址:3000`
   - 例如：`http://192.168.1.100:3000`

### 方法二：本地访问（仅本机）

- 直接访问：`http://localhost:3000`

## 多用户同时使用

✅ **系统支持多用户同时登录和使用**
- 每个用户使用独立的账号登录
- 数据会实时同步到服务器
- 所有用户可以看到和编辑客户记录
- 离线编辑的数据会在联网后自动同步

## 安全建议

⚠️ **首次使用后，建议修改默认密码**

目前系统使用默认密码，建议：
1. 登录后尽快修改密码（需要添加修改密码功能）
2. 或者为每个同事分配不同的账号
3. 定期备份数据库文件：`server/data/csm.db`

## 账号管理

如果需要添加更多账号或修改账号信息，可以：
1. 使用管理员账号登录
2. 联系系统管理员添加新账号
3. 或直接修改数据库（需要技术知识）

## 注意事项

- 所有用户共享同一个数据库
- 数据会实时同步，请谨慎操作删除功能
- 建议定期备份数据
- 如果服务器关闭，同事将无法访问（除非使用离线模式）
