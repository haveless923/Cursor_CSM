```

### 步骤2：获取你的IP地址

你的电脑IP地址是：**172.20.10.2**

（如果IP地址变化，可以在终端运行：`ifconfig | grep "inet " | grep -v 127.0.0.1`）

让同事在浏览器输入：

```

## 🔐 登录步骤

1. 打开浏览器访问系统地址
2. 输入分配的账号和密码
3. 点击"登录"
4. 登录成功后即可使用系统

## ⚠️ 重要提示

1. **服务器必须运行**：只有你的电脑上运行了 `npm run dev`，同事才能访问
2. **同一网络**：同事需要和你在同一个局域网（WiFi/网络）下
3. **防火墙**：如果无法访问，可能需要允许 3000 和 3001 端口的访问
4. **数据共享**：所有用户共享同一个数据库，数据会实时同步

## 🔧 如果无法访问

### 检查1：服务器是否运行
```bash
# 检查端口是否在监听
lsof -ti:3000,3001
```

### 检查2：防火墙设置
- macOS: 系统设置 > 网络 > 防火墙
- 确保允许 Node.js 的网络访问

### 检查3：IP地址是否正确
```bash
# 重新获取IP地址
ifconfig | grep "inet " | grep -v 127.0.0.1
```
